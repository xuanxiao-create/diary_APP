# 日环记APP - 功能界面静态网站

## 项目简介

这是一个基于原生日环记APP功能创建的功能界面静态网站版本，提供了完整的任务管理、计划制定、回顾记录和数据统计功能。在原有功能基础上，新增了用户角色系统、自定义分类管理、时间作息表、任务时间段等强大功能。

## 使用说明

### 1. 访问网站
打开浏览器访问：`http://localhost:8889`

### 2. 测试账号
- **管理员账号**: `admin` / `admin123`
- **测试用户**: `demo` / `123456`
- **游客模式**: 点击"游客模式"按钮即可进入

### 3. 功能体验
1. **注册新用户**或使用测试账号登录
2. **创建任务** - 在任务页面添加新的待办事项（支持时间段选择）
3. **制定计划** - 在计划页面创建日/周/月计划
4. **写回顾** - 在回顾页面记录每日总结
5. **查看统计** - 在统计页面查看完成率数据
6. **管理分类** - 在分类页面创建自定义任务分类
7. **作息表管理** - 在作息表页面制定时间安排
8. **管理员功能** - 使用admin账号体验用户管理功能

## 部署说明

这是一个纯静态网站，可以部署到任何静态托管服务：

1. **GitHub Pages**: 将文件推送到GitHub仓库，启用Pages功能
2. **Netlify**: 直接拖拽文件夹到Netlify部署
3. **Vercel**: 连接GitHub仓库自动部署
4. **传统服务器**: 上传到Web服务器目录

## 开发说明

### 本地开发
```bash
# 启动本地服务器
python3 -m http.server 8889

# 或使用Node.js
npx serve .
```
